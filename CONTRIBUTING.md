# Contributing to NotebookLM Wizard Agent

First off, thank you for considering contributing to this project! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
- Check the [Issues](https://github.com/xlastfire/notebooklm/issues) to see if the bug has already been reported.
- If not, open a new issue using the **Bug report** template.

### Suggesting Enhancements
- Check the [Issues](https://github.com/xlastfire/notebooklm/issues) to see if the feature has already been suggested.
- Open a new issue using the **Feature request** template.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes (`python verify_lang.py`).
4. Make sure your code lints.
5. Issue that pull request!

## Style Guide

- We use `rich` for all CLI output.
- Follow PEP 8 for Python code.
- Use meaningful commit messages.
